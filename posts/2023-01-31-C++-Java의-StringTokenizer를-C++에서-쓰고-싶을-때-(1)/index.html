[C++] Java의 StringTokenizer를 C++에서 쓰고 싶을 때

게시 2023/01/31
By 김동주
2 분읽는 시간

전역 변수로 구조체 변수 StringTokenizer stringTokenizer를 선언해놓고 사용한다.

구조체 StringTokenizer에는 다음 3가지 메소드를 구현한다.

  • void readline(): stdin으로 부터 한 줄을 읽어와 토큰을 파싱할 준비를 한다.
  • char* nextToken(): 구분자(공백 문자), 혹은 문자열이 끝나는 것을 기준으로 문자열을 나누어 차례차례 반환한다. 한 번 반환된 토큰은 다시 반환되지 않는다.
  • bool hasNext(): 다음에 가져올 토큰이 있는지를 알려준다.

구현 내용은 다음과 같다:

#include <iostream>

#define BUFFER_CAPACITY 32

struct StringTokenizer {
	char buffer[BUFFER_CAPACITY];
	char *ptr;

	void readline() {
		std::cin.getline(buffer, BUFFER_CAPACITY, '\n');
		ptr = buffer;
	}

	char *nextToken() {
		char *retval = ptr;
		while (*ptr != ' ' && *ptr != '\0') {
			ptr++;
		}
		*ptr = '\0';
		ptr++;
		return retval;
	};

	bool hasNext() {
		return *ptr != '\0';
	};
} stringTokenizer;

// ...


// 사용 예시

#include <cstring>

int main(int argc, char **argv) {
    char str[];
    char c;

    stringTokenizer.readline();

    strcpy(str, stringTokenizer.nextToken());

    if (stringTokenizer.hasNext()) {
        c = *stringTokenizer.nextToken();
    }
}
  • c언어 문자열에서는 null문자인 '\0'을 만나면 문자열이 끝났음으로 인식한다는 점을 이용했다.
  • 멤버 ptr은 항상 토큰의 시작점을 가리킨다. nextToken()을 호출 할 때 마다, ptr이 가리키는 현재 토큰을 반환하고 ptr을 다음 토큰의 시작점으로 전진 시킨다.
  • 다음 토큰이 없다는 것은 입력 받은 문자열을 전부 살펴봤음을 의미한다. hasNext()에서는 다음 토큰이 있는지를 알기위해 ptr이 가리키는 주소의 값이 \0인지를 살펴본다.
